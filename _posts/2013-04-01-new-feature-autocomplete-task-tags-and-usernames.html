We've added a new feature in with task tags and @Username tagging in Sandglaz.

The next time you tag a task, you'll get an auto-complete dropdown showing all other tags you've used on the grid before.
This makes it faster and easier to use repeat tags on tasks.

If you're using an @Username tag to assign tasks to teammates on a shared grid, you'll also see an auto-complete dropdown menu showing all @Usernames that your grid is shared with.
You don't have to remember @Usernames of coworkers or teammates you've shared grids with.

Make sure you all shared users of a grid set their @Username.

Try it out and let us know how you like it in the comments below.
