We’re happy to let you know that you can now attach files to your Sandglaz tasks using Google Drive. This feature complements the Dropbox attachment feature we released earlier this year.

How to attach files to your tasks

Similarly to Dropbox attachments, attaching files via Google Drive is as easy as opening the task detail view and clicking on the ‘Choose from Google Drive' button.

 

You will get a Google Drive file selector, where you can select the files you want to add to your task.

 

 

The selected files are then attached to the file, and can be accessed by your team members. The files open in Google Drive, so you will be all collaborating on the same document.

 

A file attachment label appears next to tasks with attached files. This way you will be able to tell if there are files attached without opening the task detail view.

That’s about as simple as it gets! You can visit Sandglaz right away to start attaching Google Drive files to your tasks.
